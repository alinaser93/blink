طوّر **شاشة العنوان والخريطة** لتشبه تطبيق Blinkit.

الملف المستهدف: `src/customer/AddressMapScreen.jsx` (تحديد الموقع، العناوين المحفوظة).

التعليمات:
1. اقرأ الملف بالكامل أولاً، واقرأ `CLAUDE.md` للسياق والقواعد.
2. الهدف: نمط Blinkit — أقل احتكاك، عنوان بضغطة، RTL، خط Cairo، «د.ع»، أخضر `#0C831F`، خصائص CSS منطقية.
3. لا تكسر الميزات الشغّالة، وحافظ على تكامل التطبيق (نفس `src/lib`).
4. بعد التعديل: `npm run build` وتأكّد لا أخطاء.
5. لا تدفع ولا تنشر على `main` إلا حين أطلب صراحةً.
