طوّر **الصفحة الرئيسية للزبون** لتشبه تطبيق Blinkit.

الملف المستهدف: `src/customer/HomePage.jsx` (شبكة الأقسام، أرفف المنتجات، «الأكثر مبيعاً»، الترويسة، التبويبات).

التعليمات:
1. اقرأ الملف بالكامل أولاً، واقرأ `CLAUDE.md` للسياق والقواعد.
2. الهدف: نمط Blinkit — نظيف، RTL، خط Cairo، عملة «د.ع»، أخضر `#0C831F`، **بلاطات منتجات بإيموجي** عبر `emojiFor` من `src/customer/emoji.js`، وخصائص CSS منطقية فقط.
3. لا تكسر الميزات الشغّالة، وحافظ على تكامل التطبيق (نفس `src/lib` و Supabase والسلّة المشتركة `cart.jsx`).
4. بعد التعديل: شغّل `npm run build` وتأكّد لا أخطاء.
5. لا تدفع للفرع ولا تنشر على `main` إلا حين أطلب صراحةً.
