طوّر **تطبيق المندوب** لتشبه نمط Blinkit للمناديب.

الملف المستهدف: `src/rider/RiderApp.jsx` (الطلبات المعيّنة، الملاحة، تأكيد الاستلام/التوصيل).

التعليمات:
1. اقرأ الملف بالكامل أولاً، واقرأ `CLAUDE.md` للسياق والقواعد.
2. الهدف: واجهة مندوب نظيفة وسريعة، RTL، خط Cairo، «د.ع»، أخضر `#0C831F`، خصائص CSS منطقية.
3. **التواصل مع لوحة الأدمن عبر حالة الطلب**: تأكيد التوصيل يحدّث `orders.status` عبر `setOrderStatus` في `src/lib/api.js`، فتراه اللوحة (مهيّأ لـ Realtime). لا تكسر هذا الترابط.
4. لا تكسر الميزات الشغّالة، وحافظ على تكامل التطبيق.
5. بعد التعديل: `npm run build` وتأكّد لا أخطاء.
6. لا تدفع ولا تنشر على `main` إلا حين أطلب صراحةً.

المطلوب الآن: $ARGUMENTS
