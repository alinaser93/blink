طوّر **صفحة تتبّع الطلب** لتشبه تطبيق Blinkit.

الملف المستهدف: `src/customer/OrderTrackingScreen.jsx` (الخريطة، المندوب، ETA، مراحل الطلب).

التعليمات:
1. اقرأ الملف بالكامل أولاً، واقرأ `CLAUDE.md` للسياق والقواعد.
2. الهدف: نمط Blinkit — عدّاد وصول، مراحل واضحة، RTL، خط Cairo، «د.ع»، أخضر `#0C831F`، خصائص CSS منطقية.
3. لا تكسر الميزات الشغّالة، وحافظ على تكامل التطبيق (حالة الطلب من `orders` عبر `src/lib/api.js`).
4. بعد التعديل: `npm run build` وتأكّد لا أخطاء.
5. لا تدفع ولا تنشر على `main` إلا حين أطلب صراحةً.
