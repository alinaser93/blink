طوّر **صفحة تفاصيل المنتج** لتشبه تطبيق Blinkit.

الملف المستهدف: `src/customer/ProductPage.jsx` (المعرض، الوصف، السعر، المنتجات المشابهة، شريط الإضافة).

التعليمات:
1. اقرأ الملف بالكامل أولاً، واقرأ `CLAUDE.md` للسياق والقواعد.
2. الهدف: نمط Blinkit — نظيف، RTL، خط Cairo، «د.ع»، أخضر `#0C831F`، **بلاطات إيموجي** عبر `emojiFor` من `src/customer/emoji.js`، خصائص CSS منطقية.
3. لا تكسر الميزات الشغّالة، وحافظ على تكامل التطبيق (نفس `src/lib` و Supabase).
4. بعد التعديل: `npm run build` وتأكّد لا أخطاء.
5. لا تدفع ولا تنشر على `main` إلا حين أطلب صراحةً.
