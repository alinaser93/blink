طوّر **صفحة السلة والدفع** لتشبه تطبيق Blinkit.

الملف المستهدف: `src/customer/CartPage.jsx` (عناصر السلة، الفاتورة، الكوبون، العنوان، طرق الدفع، شاشة النجاح). والسلّة المشتركة `src/customer/cart.jsx`.

التعليمات:
1. اقرأ الملف بالكامل أولاً، واقرأ `CLAUDE.md` للسياق والقواعد.
2. الهدف: نمط Blinkit — تقليل احتكاك الدفع، RTL، خط Cairo، «د.ع»، أخضر `#0C831F`، **بلاطات إيموجي** عبر `emojiFor` من `src/customer/emoji.js`، خصائص CSS منطقية.
3. انتبه لحسابات الفاتورة (لا تكسرها). لا تكسر الميزات الشغّالة، وحافظ على تكامل التطبيق (نفس `src/lib` و Supabase).
4. بعد التعديل: `npm run build` وتأكّد لا أخطاء.
5. لا تدفع ولا تنشر على `main` إلا حين أطلب صراحةً.

المطلوب الآن: $ARGUMENTS
