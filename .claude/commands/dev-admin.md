طوّر **لوحة الإدارة (الأدمن)** لتشبه نمط Blinkit الإداري.

الملف المستهدف: `src/admin/AdminDashboard.jsx` (الرئيسية، الطلبات الحية/كانبان، المنتجات والمخزون، الأقسام، العملاء، التقارير).

التعليمات:
1. اقرأ الملف بالكامل أولاً، واقرأ `CLAUDE.md` للسياق والقواعد.
2. الهدف: لوحة نظيفة احترافية، RTL، خط Cairo، «د.ع»، أخضر `#0C831F`، خصائص CSS منطقية.
3. البيانات عبر `src/lib/api.js` (تعمل في وضعَي «متصل بـ Supabase» و«تجريبي»). أي تعديل على سلوك البيانات = دالة في `api.js` (+ توثيق السكيمة في `supabase/schema.sql` عند اللزوم).
4. لا تكسر الميزات الشغّالة، وحافظ على تكامل التطبيق.
5. بعد التعديل: `npm run build` وتأكّد لا أخطاء.
6. لا تدفع ولا تنشر على `main` إلا حين أطلب صراحةً.
> ملاحظة: لا يوجد دور «تاجر» منفصل؛ هذه لوحة الأدمن/مدير الفرع.

المطلوب الآن: $ARGUMENTS
